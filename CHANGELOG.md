# Changelog

## 0.1.0 — 2026-03-31

### Added

- `morphology` — Galaxy classification (Hubble types), Sersic profiles, Faber-Jackson, Tully-Fisher
- `halo` — NFW dark matter halo profiles, virial radius, concentration-mass, circular velocity
- `cosmic_web` — Tidal tensor classification (node/filament/sheet/void), density contrast, correlation function
- `power_spectrum` — Eisenstein-Hu transfer function, growth factor, σ(R), primordial power
- `constants` — Planck 2018 cosmological parameters, σ₈, n_s
- `error` — BrahmandaError with input/output validation helpers
- Integration tests, criterion benchmarks, runnable example
